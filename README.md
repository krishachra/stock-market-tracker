
# 📈 Stock Market Tracker

## 📌 Description

This project is a web application that allows users to search and view stock market data such as stock prices and trends using a public API.

## 🚀 Features

* Search stocks by company name or symbol
* Display stock price and basic details
* Sort stocks based on price or performance
* Responsive user interface

## 🔗 API Used

Alpha Vantage API (https://www.alphavantage.co/)

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (Fetch API)

## 📅 Milestone Plan

* Milestone 1: Project setup and planning
* Milestone 2: API integration
* Milestone 3: Add search, filter, and sorting features
* Milestone 4: Deployment and final submission
